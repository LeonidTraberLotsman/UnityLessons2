Этот скрипт по задумке убирает весь мусор в репозитории и возвращет последнюю версию.

Текст скрпипта:
git stash -u    		убирает в отдельную кучку все изменения, включая untracked files (новосозданные файлы)
git stash clear 		удаляет эту кучку с файлами
git reset --hard		переставляет изменения, которые кто-то зачем-то сохранил
git pull			берёт новую версию с GitHub
pause				позволяет не закрыть терминал, то есть прочитать о успешности или неуспешности операций
